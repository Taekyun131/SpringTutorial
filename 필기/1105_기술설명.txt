
1. 제목   :   장바구니 UI 

2. 구현한 기능 :	
   - 상품정보, 수량, 가격, 배송비를 입력해 주문가능
   - 장바구니에 담긴 내역 확인가능
   - 선택한 상품은 삭제 가능
   - 선택상품금액, 총 배송비, 주문금액을 계산하여 표시
   

3. 구현과정 풀이

입력창과 장바구니에 담긴 내역을 테이블로 정의한다.
상품정보와 수량, 상품금액, 배송비를 입력하고 주문하기 버튼을 누르면
장바구니 내역 테이블에 행과 열을 추가해 체크박스와 입력한 정보가 표시된다.
체크박스를 선택하고 삭제버튼을 누르면  선택한 상품들의 정보가 포함된 행을 삭제한다.
장바구니 테이블의 thead태그의 체크박스를 선택하면 
테이블 모든 행의 체크박스가 선택되고, 반대로
thead태그의 체크박스를 선택해제하면 모든 행의 체크박스가 선택해제된다.
장바구니에 담긴 상품의 상품금액*수량 과 배송비를 각각 합산해 선택상품금액과 총 배송비를 계산하고, 
선택상품금액과 총 배송비의 합을 주문금액으로 계산한다. 테이블의  행이 추가되거나 삭제될 때 계산을 진행한다.



4. 주요사용기술
정적인 html문서에 동적인 요소를 추가하기 위해 자바스크립트를 활용했으며
html의 DOM구조를 통해 html문서의 각 요소들을 제어했다.  
addRow()와 delRow()메서드에서는 onclick을 사용해 클릭 이벤트를 
테이블 객체의 insertRow()와 deleteRow() 메서드로 처리해 장바구니 담기와 선택상품 삭제를 구현했다. 
calMoney() 메서드에서는 테이블의 행과 열의 입력값을 Number()메서드를 활용해 정수형으로 변환한 다음,  
상품금액과 배송비를 계산해 표시했다.
allChk1() 메서드는 onchange를 사용해 이벤트를 처리했으며, 체크박스 전체선택과 전체선택해제를 구현했다.
